# Student Management System (SQL Mini Project)

## Project Description
This project is a SQL-based Student Management System designed to perform CRUD operations,
apply normalization, and retrieve data using joins.

## Features
- Student and Course management
- CRUD operations
- SQL Joins
- Normalized database (3NF)
- Views and Stored Procedures

## Technologies Used
- SQL
- MySQL / Oracle SQL

## Folder Structure
- database: Database and table creation scripts
- queries: CRUD and join queries
- procedures: Stored procedures
- views: SQL views
- diagrams: ER diagram
- screenshots: Output screenshots
